# 多端前端搭建

这部分负责 `Web / 微信小程序 / Android`。

## 1. 为什么这里选 uni-app

因为你的 iOS 已经独立使用 `SwiftUI`，所以另外三端最适合共享一套前端代码：

- `Web`
- `微信小程序`
- `Android`

`uni-app` 的优势是：

- API 与小程序比较贴近
- 可以直接用 `uni.request`
- 对国内多端场景支持比较友好

## 2. 当前前端目录

- `src/pages/home`: 智能出题页
- `src/pages/history`: 历史题集页
- `src/pages/detail`: 题集详情页
- `src/api`: 请求封装
- `src/utils`: 格式化工具

## 3. API 请求在哪

你可以重点看这两个文件：

## 4. 本地运行

### 安装依赖

```bash
cd /Users/geraldgan/Documents/GeraldGan/实践/InterviewSystem/frontend
npm install
```

### 启动 H5

```bash
npm run dev:h5
```

如果你只是想先在自己电脑浏览器里看页面，保持后端启动后直接访问终端输出里的本地地址即可。

### 构建 H5

```bash
npm run build:h5
```

### 启动微信小程序

```bash
npm run dev:mp-weixin
```

如果你只是想先生成一份可导入微信开发者工具的静态产物，也可以执行：

```bash
npm run build:mp-weixin
```

## 5. 接口地址怎么改

通过环境变量控制：

- [frontend/.env.example](/Users/geraldgan/Documents/GeraldGan/实践/InterviewSystem/frontend/.env.example)
- [frontend/.env.local](/Users/geraldgan/Documents/GeraldGan/实践/InterviewSystem/frontend/.env.local)

默认值：

```bash
VITE_API_BASE_URL=http://127.0.0.1:8081
```

你当前这台电脑已经帮你补好了本地文件：

```bash
VITE_API_BASE_URL=http://10.0.20.45:8081
```

这意味着：

- `H5` 在你自己的电脑浏览器里运行也能访问后端
- `微信小程序` 在真机预览时也能访问到你电脑上的 Spring Boot 服务

如果你以后换了 Wi-Fi，这个 IP 可能会变化，到时同步更新 [frontend/.env.local](/Users/geraldgan/Documents/GeraldGan/实践/InterviewSystem/frontend/.env.local) 即可。
